🎓 AI Study Assistant

An AI-powered study assistant built using Python, Gradio, and Hugging Face's Mistral-7B-Instruct model to help students learn more effectively.

🚀 Features

- 📚 Topic Explanation
  
  - Get simple and easy-to-understand explanations for any topic.
  - Uses analogies and beginner-friendly language.

- 📝 Quiz Generator
  
  - Automatically generates 5 multiple-choice questions (MCQs) on any subject.
  - Includes answers for self-assessment.

- 📋 Notes Summarizer
  
  - Converts lengthy notes into concise revision-friendly bullet points.
  - Helps students prepare quickly for exams.

- 🎨 Interactive User Interface
  
  - Built with Gradio for a clean and user-friendly experience.
  - Tab-based navigation for easy access to features.

🛠️ Tech Stack

- Python
- Gradio
- Hugging Face Inference API
- Mistral-7B-Instruct-v0.2

⚙️ Installation

1. Clone the Repository

git clone https://github.com/DivyanshSingh9073/AI-Study-Assistant-.git
cd AI-Study-Assistant-

2. Install Dependencies

pip install -r requirements.txt

3. Set Up Hugging Face Token

Create an environment variable:

export HF_TOKEN="your_huggingface_token"

For Windows:

set HF_TOKEN=your_huggingface_token

Get your token from:
https://huggingface.co/settings/tokens

4. Run the Application

python app.py

The application will launch locally and provide a shareable Gradio link.
